## Title

Simple Board — Native macOS Onboarding Built with Codex + GPT-5.6

## Description

Simple Board is a native macOS onboarding workspace for growing teams. Owners create structured programs, assign employees, inspect progress and learning evidence, and identify people who may need support. Employees receive one focused journey covering video, reading, checklist, quiz, task, document, and meeting materials.

In this demo, Budi completes the Kitchen Code of Conduct and moves from 71% to 86%, immediately closing the loop back to the owner dashboard and Performance view.

We built Simple Board with Codex and GPT-5.6. Codex worked directly in the repository to help convert a browser prototype into a native SwiftUI app, implement typed models and persistence, diagnose layout issues from screenshots, run Xcode builds and tests, integrate the employee journey, and prepare the TestFlight release. GPT-5.6 helped reason across product intent, permissions, local-first storage, secure employee access, responsive UX, and end-to-end test coverage.

Source code: https://github.com/abadikaka/simpleboard

Narration disclosure: This video uses AI-generated narration created with OpenAI speech synthesis. The ambient bed and interface tones were synthesized specifically for this video; no stock music or sampled third-party audio was used.
